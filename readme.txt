Adapt WooCommerce to work with FactuSol.

== Description ==
[youtube http://www.youtube.com/watch?v=qSiVVJ4DeyI]

Create fields to import FactuSol data:

  - Product ID
  - Product Category ID
  - Order ID

== Installation ==
Install like any other WordPress plugin.  

[Installing Plugins](https://codex.wordpress.org/Managing_Plugins)

== Frequently Asked Questions ==

No, minumun FactuSol version required is FactuSol 2017 EI
